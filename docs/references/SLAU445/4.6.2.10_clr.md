# 4.6.2.10 CLR

- \* `CLR[.W]` - Clear destination
- \* `CLR.B` - Clear destination

## Syntax 

- `CLR dst`
- `CLR.W dst`
- `CLR.B dst`

## Operation 

0 â†’ dst

## Emulation

- `MOV #0, dst`
- `MOV.B #0, dst`

## Description

The destination operand is cleared.

## Status Bits

Status bits are not affected.

## Example 1

RAM word TONI is cleared.

```asm
CLR  TONI  ; 0 -> TONI
```

## Example 2

Register R5 is cleared.

```asm
CLR R5
```

## Example 3

RAM byte TONI is cleared.

```asm
CLR.B  TONI  ; 0 -> TONI
```
